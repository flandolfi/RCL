Assignment - Week 9
===================

Conference
----------

Project files:

 - `Client.java`;
 - `Conference.java`;
 - `ConferenceDays.java`;
 - `Server.java`.


Use the commands below to compile the project:

    $ javac Server.java
    $ javac Client.java

Run the server with the command

    $ java Server

and then any number of clients with

    $ java Client
