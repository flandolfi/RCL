Assignment - Week 10
====================

Forum
-----

Project files:

 - `Client.java`;
 - `Forum.java`;
 - `ForumSubscriber.java`;
 - `RemoteForum.java`;
 - `Server.java`;
 - `Subscriber.java`.


Use the commands below to compile the project:

    $ javac Server.java
    $ javac Client.java

Run the server with the command

    $ java Server

and then any number of clients with

    $ java Client
